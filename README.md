# Calm Coffee Web Experience

「いつもの毎日に、深呼吸できる余白を。」をコンセプトにした、流山おおたかの森に佇む高品質なコーヒーショップ **Calm Coffee** のブランドサイトです。

## 技術スタック

- **Core**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Deployment**: Vercel

## 主な機能と仕様

### 1. ヒーローセクション
- **シームレス・クロスフェード**: 背景画像が穏やかに切り替わるアニメーション。
- **Ken Burns エフェクト**: 画像がゆっくりと拡大・移動し、奥行き感を演出。
- **パララックス**: スクロールに応じてコピーが浮き上がる視覚効果。

### 2. アニメーション演出
- **ソフトブラー・リビール**: スクロールに合わせて画像が「ぼかし」から浮かび上がる現代的な演出。
- **3D Tilt**: メニューカードにマウスを合わせると、パースペクティブ（奥行き）を伴って傾くインタラクション。

### 3. モバイル最適化（レスポンシブ）
- **スマートフォン対応**: 375px〜の最小デバイスでも崩れないレイアウト。
- **専用モバイルメニュー**: ハンバーガーボタンによるナビゲーション展開。

### 4. 模擬決済システム (Mock Commerce)
- **ショッピングカート**: 商品の追加・数量変更・削除が可能なサイドバー型カート。
- **チェックアウト・フロー**: 配送先入力から決済完了（シミュレーション）までの本格的なUXを実装。
  > ※実際の金銭的な取引は行われません。

### 5. パフォーマンス最適化
- **画像自動圧縮**: `vite-plugin-image-optimizer` を導入し、ビルド時に合計数十MBの画像を約60%（50MB以上）削減。
- **遅延読み込み (Lazy Loading)**: ブラウザ標準の lazy loading を全アセットに適用。
